# 🛒 Digi Tools Buying Website

## 📌 Project Overview
**Digi Tools Buying Website** is a responsive e-commerce style web application built using modern frontend technologies.  
Users can browse digital tools, select products, view the total selected items and total price, and proceed to checkout easily.  
The website also provides interactive toast notifications to improve user experience.  

This project demonstrates modern frontend development skills using React, Tailwind CSS, and other libraries, along with responsive design principles.

## ✨ Main Features
- **Responsive Design:** Works seamlessly across mobile, tablet, and desktop devices.  
- **Product Selection:** Users can add or remove products from the cart.  
- **Total Price Calculation:** Automatically calculates and displays the total price of selected products.  
- **Toast Notifications:** Shows interactive messages when products are added, removed, or checkout is completed.  
- **Easy Checkout:** Users can proceed to checkout with a clear summary of selected items.

## 💻 How to Run Locally

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/digi-tools.git
```

### 2️⃣ Navigate to the project folder
```bash
cd digi-tools
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Run the development server
```bash
npm run dev
```

Open in your browser:
```
http://localhost:5173
```
