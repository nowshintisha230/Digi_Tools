# Project Title: 🛒 Digi Tools Buying Website

## 📌 Project Description

**Digi Tools Buying Website** is a simple e-commerce style web application built using modern frontend technologies. Users can explore different digital tools, select the products they want, view the total selected items and total price, and proceed to checkout easily. The website also provides real-time toast notifications for better user experience.

---

## 🛠️ Technologies Used

* **HTML5**
* **Tailwind CSS**
* **React JS**
* **JSON**
* **Font Awesome**
* **DaisyUI**
* **Vite**
* **External CSS**
* **React Toast**

---

## ✨ Features

✔ **Responsive Design**
The website is fully responsive and works smoothly across mobile, tablet, and desktop devices.


✔ **Total Price Calculation**
Automatically calculates and displays the total price of selected products.


✔ **Toast Notifications**
Interactive toast messages appear when products are added to the cart.
